SLOTH project:

Superfluid Library for Operations Transformations and Hacking

To install this magnificent library you must introduce this code:

pip install git+https://github.com/gonmendoza/sloth.git

To be able to run the test files you will have to clone the repository https://github.com/gonmendoza/sloth.git

And then in the folder you have installed it, run the comand "pytest"
